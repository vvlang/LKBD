## 准备工作
可以选择我的固件，也可以刷自己构建的固件。
下载好固件使用工具刷到ProMicro中 [QMK Toolbox](https://github.com/qmk/qmk_toolbox)

## 更改键位
下载VIA工具 [VIA工具](https://github.com/the-via/releases/releases/)即可更改.
